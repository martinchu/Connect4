##Installation

1. Clone or Download the reposity. If you clone the reposity via command line, please disregard step 2.
2. Extract the zip file
3. Open Terminal
4. type
```
cd Path/to/Connect4/Directory/
```
5. type
```
make
```
6. type
```
./c4
```
and start the game!
